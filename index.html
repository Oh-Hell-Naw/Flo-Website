About

I am Florian, a student from Germany with great interest in computer science and software development. Currently, I am developing purely in Python. However, I am planning to learn more languages (Like TypeScript) in the future.

One of my biggest projects is at the same time a school project which is the Dijkstra Demo program. It is a program to visualize the Dijkstra algorithm in action. The program is written in Python and uses the PyQt5 framework.

Next is my physics engine which is also written in Python. It is a simple physics engine which can simulate gravity and collisions aswell as other forces. It is based on the matplotlib library which is used to visualize the simulation.

What am I working on?

As of right now I am working on training an AI for Geometry Dash levels. Currently I am in early development trying to prepare everything for the model training. So far I have successfully decrypted and extracted necessary data from the game files to decrypt any level.
